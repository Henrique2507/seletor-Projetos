Conteudos



O TML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo da web. Ele oferece novas funcionalidades e elementos semânticos para aprimorar a experiência do usuário e a eficiência do desenvolvimento web.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS


Em termos gerais, um display é qualquer superfície que apresenta informações visuais, como imagens, textos ou vídeos, para um observador. É o meio pelo qual dispositivos eletrônicos comunicam visualmente com o usuário.

Display Caracteristicas
Block Elementos com display BLOCK, também com um nome bem semântico, são elementos que se comportam como um “bloco” na página, cuja largura sempre vai ocupar 100% da largura do elemento-pai
Inline Em programação, código inline (ou "embutido") refere-se a código ou dados que são inseridos diretamente no local onde são usados, em vez de serem chamados de uma localização separada.
Inline-Block O display: inline-block em CSS faz com que um elemento seja tratado como um bloco, mas ainda mantenha o comportamento de um elemento inline, permitindo que vários elementos sejam exibidos lado a lado e que propriedades como largura e altura sejam aplicadas.
Flex define como um item (chamado de flex item) se comporta em relação ao espaço disponível dentro de seu contêiner, permitindo que cresça para preencher o espaço ou encolha conforme necessário.
